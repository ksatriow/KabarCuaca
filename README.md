## Specs & Open-source libraries
- Minimum SDK 16
- 100% Kotlin based
- TMDB API
- MVVM Architecture
- Architecture Components (Lifecycle, LiveData, ViewModel, Room Persistence)
- Room
- Kodein dependency injection
